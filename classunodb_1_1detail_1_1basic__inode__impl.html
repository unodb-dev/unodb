Detailed Description

template<class ArtPolicy>
class unodb::detail::basic_inode_impl< ArtPolicy >

Base class for all internal node types.

Extends common header and defines methods shared by all internal nodes. The header type is specific to thread-safety policy: for OLC, it includes lock and version tag metadata.

Contains generic internal node code: key prefix, children count, and dispatch for add/remove/find operations to specific node types.
