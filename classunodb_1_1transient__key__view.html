Detailed Description

Non-owning view of key bytes with scoped lifetime.

Returned by iterator get_key() when the key is reconstructed from the inode path (full_key_in_inode_path trees). The view is valid only until the next iterator movement (next/prior/seek/invalidate).

Does NOT allow implicit construction from key_view, preventing accidental storage of a key_view where a transient_key_view is expected. Does NOT implicitly convert to key_view — use .view() for explicit access.
