Detailed Description

template<bool Enabled, class Storage>
struct unodb::detail::value_bitmask_field< Enabled, Storage >

A bitmask that tracks which child slots hold packed values rather than pointers.

When Enabled is false the struct is empty and all operations compile away, so inodes pay zero overhead for db types that do not use value-in-slot.
