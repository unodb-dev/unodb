Detailed Description

template<typename Key, typename Value>
class unodb::detail::inode_4< Key, Value >

Internal node with 4 children for non-thread-safe ART.

Smallest internal node type, used when a leaf needs to be split or when a larger node shrinks below minimum occupancy.
