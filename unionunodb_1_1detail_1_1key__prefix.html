Detailed Description

template<typename ArtKey, template< class > class CriticalSectionPolicy>
union unodb::detail::key_prefix< ArtKey, CriticalSectionPolicy >

Key prefix for prefix compression in internal nodes.

Stores zero or more bytes that are common prefix shared by all children of a node, supporting prefix compression in the index.

Template Parameters
ArtKeyInternal ART key type
CriticalSectionPolicyPolicy for thread-safe access
