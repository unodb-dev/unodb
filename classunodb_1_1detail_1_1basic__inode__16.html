Detailed Description

template<class ArtPolicy>
class unodb::detail::basic_inode_16< ArtPolicy >

Internal node with 5-16 children (N16).

Like in basic_inode_4, keys are maintained in sorted order and child pointers correspond 1:1 with key positions. Uses SIMD (SSE2/AVX2/NEON) for optimized key search when available.
