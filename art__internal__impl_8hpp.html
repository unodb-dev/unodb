Detailed Description

Implementation details for Adaptive Radix Tree (ART) internal nodes.

Provides node implementations (basic_inode_4, basic_inode_16, basic_inode_48, basic_inode_256), leaf storage, key prefix compression, and supporting infrastructure for both single-threaded and optimistic lock coupling (OLC) variants. This header is not part of the public API.
