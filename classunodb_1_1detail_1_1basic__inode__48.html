Detailed Description

template<class ArtPolicy>
class unodb::detail::basic_inode_48< ArtPolicy >

Internal node with 17-48 children (N48).

Uses 256-byte child_indexes array directly indexed by key byte, storing indices into 48-element child pointer array. Neither child_indexes nor child pointers are dense. Uses SIMD for finding first empty slot.
